# 投稿案生成用 入力テンプレート

このファイルに元となる文章を記入してください。

## 入力文章

ここに、SNS投稿案を生成したい元となる文章を記入してください。

以下のような内容を含めると、より良い投稿案が生成されます：

### 推奨される内容
- **稀有な要素**: あなたの独自の経験、視点、立場
- **具体的なエピソード**: 数字や固有名詞を含む具体的な話
- **感情**: 喜び、驚き、怒り、悲しみなどの感情が含まれる内容
- **プロセス**: 現在進行中の挑戦や成長過程
- **問いかけ**: 読者が考えたくなるような問い

## 入力例1: ビジネス・起業系

```
3年前、月収20万円のサラリーマンだった私が、今では年商3億円の会社を経営しています。

きっかけは、上司に理不尽に怒られた日でした。「お前には才能がない」と言われ、悔しさで眠れない夜を過ごしました。

その日から、毎日3時間の副業を始めました。最初の半年は全く稼げず、月の収入は3000円程度。妻からは「いつまで続けるの？」と冷たい目で見られる日々。

でも、諦めませんでした。

7ヶ月目、初めて月10万円を突破。その瞬間、「これはいける」と確信しました。

今では50人のチームを率い、リモートワークで世界中を旅しながら働いています。あの日、上司に怒られなければ、今の私はいなかったと思います。
```

## 入力例2: 自己啓発・学び系

```
私は30歳まで、本を1冊も読んだことがありませんでした。

活字が苦手で、学生時代は漫画しか読まなかった。社会人になってからも、YouTubeとNetflixばかりの日々。

でも、ある日、尊敬する先輩から「お前、このままで本当にいいのか？」と言われ、ハッとしました。

そこから1年間、毎日1冊の本を読むことを決意。最初は10ページで挫折しそうになりましたが、「まずは1ページだけ」と自分に言い聞かせて続けました。

1年後、私の人生は完全に変わりました。

読んだ本は365冊。年収は1.5倍に増え、人間関係も劇的に改善。何より、自分に自信が持てるようになりました。

読書は、人生を変える最強のツールです。
```

## 入力例3: 社会問題・時事ネタ系

```
日本の教育制度は、100年前からほとんど変わっていません。

みんな同じ制服を着て、同じ教室で、同じ内容を学ぶ。個性を殺し、従順な労働者を育成するシステムが、今も続いています。

でも、時代は変わりました。

AIが台頭し、暗記は無意味になりました。正解のない問いに答える力、創造性、批判的思考が求められる時代です。

なのに、学校では相変わらず「先生の言うことを聞きなさい」「正解を覚えなさい」と教えている。

これで本当に、子どもたちの未来は明るいのでしょうか？

私たちは今、教育を根本から見直す必要があります。
```

## 使用方法

1. このテンプレートをコピーして、新しいファイルを作成
2. 「入力文章」セクションに元となる文章を記入
3. Claude Codeで以下のコマンドを実行：
   ```
   /generate-sns-posts
   ```
4. このファイルをアップロードまたは内容をコピペ
5. 投稿案が生成されます

---

## 注意事項

- 文章の長さに制限はありませんが、200〜500文字程度が推奨です
- 具体的なエピソードや数字を含めると、より魅力的な投稿案が生成されます
- 感情や葛藤が含まれる内容は、バズる可能性が高まります
