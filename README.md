# SNS投稿作成ツール

バズる可能性の高いSNS投稿案を自動生成するAI駆動型ツール

## 概要

ユーザーが提供した文章を元に、𝕏（旧Twitter）とThreads向けの投稿案を各10個、合計20個自動生成します。

### 生成される投稿案の特徴
- **バズの法則に基づく**: 稀有性、プロセスエコノミー、感情トリガーなど、SNSでバズるための要素を組み込み
- **多様なトーン**: 真面目、カジュアル、ユーモラス、共感型など、様々なアプローチ
- **プラットフォーム最適化**: 𝕏は280文字以内、Threadsはストーリー性重視

## 使い方

### 方法1: スラッシュコマンドを使用（推奨）

1. Claude Codeで以下のコマンドを実行：
   ```
   /generate-sns-posts
   ```

2. 元となる文章を入力またはファイルをアップロード

3. Claudeが投稿案を生成

4. 生成された投稿案を確認後、保存を指示

### 方法2: 直接プロンプトで依頼

1. Claude Codeで以下のように依頼：
   ```
   以下の文章を元に、𝕏とThreads用の投稿案を各10個生成してください。

   [ここに元となる文章を貼り付け]
   ```

2. 生成された投稿案を確認

3. 保存が必要な場合は、Claudeに保存を依頼

## バズる要素の定義

### 1. 稀有性
「全員が知っているが、実際に出会うことはない」要素を武器にする
- 例：養蜂家→「10万匹の蜂と暮らす男」

### 2. 最強の方程式「稀有性 × 一般質問」
非日常的な存在に、誰もが共感できる日常的な質問をぶつける

### 3. プロセスエコノミー
完成品より、未完成なリアルを見せる
- 「弱者」からのスタート
- リアルタイム性

### 4. 冒頭2秒の感情トリガー
- 共感性羞恥
- 恐怖・緊張
- 驚き・ギャップ
- 共感・自分ごと化
- 感動・応援
- 怒り・不快

### 5. PR感のないPR
広告だと気づかれないまま、価値を刷り込む

### 6. 異常値を出す
AIには出せない、尖ったアイデアを提供

## 出力ファイル形式

生成された投稿案は以下の形式で保存されます：

### MDファイル
- ファイル名: `YYYYMMDD_投稿案.md`
- 内容: 見やすく整形された投稿案一覧
- 各投稿案に文字数、バズの法則、感情トリガーを明記

### TSVファイル
- ファイル名: `YYYYMMDD_投稿案.tsv`
- 内容: タブ区切り形式（Excelやスプレッドシートに貼り付け可能）
- カラム: No. | Platform | 投稿内容 | 文字数 | バズの法則 | 感情トリガー

## Pythonスクリプトの使用（オプション）

投稿案を手動で保存したい場合：

```bash
python save_posts.py [JSONファイルパス]
```

### JSONフォーマット例

```json
{
  "x_posts": [
    {
      "content": "投稿内容",
      "char_count": 50,
      "buzz_rule": "稀有性",
      "emotion_trigger": "共感"
    }
  ],
  "threads_posts": [
    {
      "content": "投稿内容",
      "char_count": 100,
      "buzz_rule": "プロセスエコノミー",
      "emotion_trigger": "感動"
    }
  ]
}
```

## 技術スタック

- **実行環境**: Claude Code
- **生成AI**: Claude (Sonnet 4.5)
- **出力フォーマット**: Markdown, TSV
- **スクリプト言語**: Python 3

## 技術的制約

- **𝕏用**: 280文字以内厳守
- **ハッシュタグ**: 使用しない
- **絵文字**: 使用しない
- **Threads用**: 𝕏より長めでOK
